/client

/client/device get post put delete

Api for getting, creating, updating, deleting metadata on a specific device

get

Retrieve metadata on a specific device

post

Create metadata for a new device

put

Update metadata for a specific device

delete

Delete metadata for a specific device

get /client/device

Retrieve metadata on a specific device

  • Request
  • Response

Query Parameters

  • device: required (string)

    device name to get metadata for

HTTP status code 200

Body

Media type: application/json

Type:

{
  "name": "name of device",
  "platform": "platform of device",
  "mac_address": "deivce's MAC",
  "ip_address": "deivce's IP address"
}

Example:

{
  "name": "test1",
  "platform": "pi",
  "mac_address": "00:0a:95:9d:68:16",
  "ip_address": "127.0.0.1"
}

  • Response

get

Retrieve metadata on a specific device

get /security/scanDevicePort

Retrieve metadata on a specific device

  • Request
  • Response

Query Parameters

  • device: required (string)

    device name to get metadata for

  • portNo: required (string)

    port number

HTTP status code 200

Body

Media type: application/json

Type:

{
  "name": "name of device",
  "ip_address": "deivce's IP address",
  "port": "port number",
  "tcp_port_closed" : "port open/closed status"
}

Example:

{
  "name": "test1",
  "ip_address": "127.0.0.1",
  "port": 9000,
  "tcp_port_closed": true
}
